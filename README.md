# r4ndsen / sqlite-php
Provides functionality for dealing with sqlite databases

## [Documentation](docs/index.md)

## Installation

via Composer
``` bash
$ composer require r4ndsen/sqlite-php
```

via GitHub
``` shell
$ git clone https://github.com/r4ndsen/sqlite-php.git
```

## Testing
``` bash
$ make tests
```
